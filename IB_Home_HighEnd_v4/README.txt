
IB Home - High-End v4 Package
----------------------------

Run locally:
1. Install Node.js (16+)
2. unzip the package
3. cd into the project folder
4. run: npm install
5. run: npm start
6. open http://localhost:3000

This package includes:
- Express backend (server.js)
- /api/products endpoint returning products (English names, Albanian descriptions)
- /api/contact endpoint (logs contact in server console)
- public/ with index.html, logo, and images folder (SVG icons)
